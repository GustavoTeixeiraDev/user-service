# User Service

Serviço responsável pela gestão de usuários no sistema.

## Tecnologias

- Node.js
- Express
- MongoDB
- Mongoose

## Endpoints

### `POST /api/users`
Cria um novo usuário.

### `GET /api/users`
Retorna todos os usuários.

### `GET /api/users/:id`
Retorna um usuário por ID.

### `PUT /api/users/:id`
Atualiza os dados de um usuário.

### `DELETE /api/users/:id`
Remove um usuário.

## Rodar localmente

```bash
npm install
npm start
